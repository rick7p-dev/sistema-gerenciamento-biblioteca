# Sistema de Gerenciamento de Biblioteca

Este projeto consiste em um sistema simples de gerenciamento de biblioteca desenvolvido em Python, com o objetivo de aplicar conceitos básicos de programação, estruturas de dados e boas práticas de desenvolvimento de software.

O sistema funciona via terminal e permite o controle de livros, usuários e empréstimos de forma organizada e intuitiva.

---

## Funcionalidades

O sistema oferece as seguintes funcionalidades:

- Adicionar novos livros à biblioteca  
- Remover livros cadastrados  
- Listar livros disponíveis e emprestados  
- Pesquisar livros pelo título  
- Registrar usuários  
- Listar usuários cadastrados  
- Registrar empréstimos de livros  
- Consultar empréstimos realizados  
- Encerrar o sistema através de menu interativo  

---

## Tecnologias Utilizadas

- Python 3  
- Pytest (para testes automatizados)  
- Git e GitHub (controle de versão)
